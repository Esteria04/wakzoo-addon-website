왁물원 애드온

왁물원과 트위치 방송을 위한 확장프로그램
다양한 기능을 만나보세요!

  • 최신 버전 다운로드
  • 더보기

주요 기능 설명

팝업 인터페이스

1. 왁물원에서 특정 게시판의 특정 페이지 좌표로 바로 갈 수 있습니다.
게시판을 선택하고, 페이지 번호를 입력한 뒤, '두재기'를 눌러보세요!
2. 트위치 채팅에서 원하는 유저의 채팅을 특정 색상으로 하이라이트 할 수 있습니다.

자동 새로고침

왁물원에 접속하면 30분마다 자동으로 새로고침해 방문수를 올려줍니다.
글을 읽고 있는데 메인화면으로 가지 않도록 하는 코드도 탑재했습니다.

트위치 보상 자동 수령

방송을 시청하다보면 나오는 채널포인트 상자를 자동으로 수령해줍니다.

  • 최신 버전 다운로드
  • 객관안 테스트 하러가기
  • 왁두상 테스트 하러가기
